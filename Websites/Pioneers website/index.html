Pioneers of Computer Science

Tim Berners-Lee

Tim Berners-Lee is a computer scientist known for creating HTML, the first web browser and inventing the world wide web.

While working at CERN, Tim Berners-Lee proposed a project to facilitate the sharing and updating of data between researchers based on the concept of hypertext. To do this he had to create the first first browser program. The first website was publish on the CERN network on 20 December 1990. From 1991, Berners-Lee collaborated with other researcher on the WorldWideWeb project which lead to the creation of the World Wide Web and the internet services we now use.

Tim Berners-Lee is now involved in advocacy and policy development. As director of the World Wide Web Foundation and the World Wide Web Consortium, he is working to ensure that the web is freely available and assessible to everyone in the world.

Important contributions

  1. 20th December 1990 - published first website
  2. On 6 August 1991, Berners-Lee posted a public invitation for collaboration with the WorldWideWeb project.
  3. 1994 -created the World Wide Web Consortium which decided that its standards should be based on royalty-free technology which could be adopted by anyone.
  4. 2009, Berners-Lee launched the World Wide Web Foundation (WWWF)
  5. Berners-Lee is a strong advocate for net neutrality
